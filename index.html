Commands

/giveaway create /giveaway create duration:1h winners:1 prize:Nitro

Create a new giveaway with customizable options including duration, winners count, prize, and requirements.

/giveaway end /giveaway end message_id:123456789

End a giveaway immediately and select winners.

Frequently Asked Questions

How do I set up the bot?

Setting up the bot is simple! Just invite it to your server using the "Invite to Server" button above. Once added, you can start creating giveaways immediately with the /giveaway create command.

What permissions does the bot need?

The bot requires the following permissions to function properly: View Channels, Send Messages, Embed Links, Add Reactions, and Manage Messages. These permissions are used to create and manage giveaways in your server.

Can I customize the giveaway messages?

Yes! You can fully customize all giveaway messages including the title, description, footer, and winner announcement. Use the /settings command to configure default messages for your server.

How does the bot prevent cheating?

Our bot uses advanced algorithms to detect and prevent common cheating methods like alt accounts and bot participation. It checks for account age, server participation history, and other factors to ensure fair giveaways.

Is there a premium version?

Yes, we offer a premium version with additional features like custom emojis, advanced analytics, scheduled giveaways, and priority support. You can upgrade through our dashboard.
