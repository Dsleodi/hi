Command Reference

All the commands you need to manage your giveaways

/giveaway create /giveaway create duration:1h winners:1 prize:Nitro

Create a new giveaway with customizable options including duration, winners count, prize, and requirements.

duration (required) How long the giveaway should last (e.g., 1h, 30m, 7d)
winners (number) Number of winners to select (default: 1)
prize (required) What the giveaway is for
/giveaway end /giveaway end message_id:123456789

End a giveaway immediately and select winners. Only available for active giveaways.

message_id (required) The ID of the giveaway message to end
/giveaway reroll /giveaway reroll message_id:123456789 winners:2

Reroll new winners for an ended giveaway. Useful if winners are unavailable.

message_id (required) The ID of the giveaway message to reroll
winners (number) Number of new winners to select (default: original amount)
/giveaway list /giveaway list active

View all active or ended giveaways in the server with their status and details.

type (string) "active" or "ended" (default: "active")
/giveaway template /giveaway template create name:weekly duration:7d

Create and manage giveaway templates for quick setup. Premium feature.

action (required) "create", "edit", "delete", or "list"
name (string) Name of the template to create/edit
/giveaway stats /giveaway stats message_id:123456789

View detailed statistics about a specific giveaway including participation rate and engagement.

message_id (required) The ID of the giveaway message to analyze
/giveaway reminder /giveaway reminder set message_id:123456789 time:1h

Set up custom reminder messages for giveaways. Premium feature.

action (required) "set" or "remove"
message_id (required) The ID of the giveaway message
time (string) When to send the reminder (e.g., "1h" for 1 hour before end)
/giveaway settings /giveaway settings color:#FF73FA

Configure default settings for giveaways in your server.

color (hex color) Default embed color for giveaways
host (user/role) Who can create giveaways (default: @everyone)

Frequently Asked Questions

Find answers to common questions about GiveawayBot

How do I set up the bot?

Setting up GiveawayBot is simple! Just invite the bot to your server using the "Invite to Server" button. Once added, you can start creating giveaways immediately with the /giveaway create command.

For advanced configuration, use /giveaway settings to set defaults for your server, or visit our web dashboard for even more control.

What permissions does the bot need?

GiveawayBot requires the following permissions to function properly:

  • View Channels
  • Send Messages
  • Embed Links
  • Add Reactions
  • Manage Messages
  • Read Message History

These permissions are used to create and manage giveaways in your server. The bot will never perform actions outside of giveaway management.

Can I customize the giveaway messages?

Yes! You can fully customize all giveaway messages including the title, description, footer, and winner announcement. Use the /giveaway settings command to configure default messages for your server.

Premium users get access to even more customization options including custom embeds with colors, thumbnails, and branded footers.

How does the bot prevent cheating?

Our bot uses advanced algorithms to detect and prevent common cheating methods:

  • Account age verification
  • Server participation history analysis
  • IP address checking
  • Behavior pattern detection
  • Role-based restrictions

You can also set custom requirements using /giveaway create requirements:... to further control who can participate.

Is there a premium version? What does it include?

Yes, GiveawayBot offers a premium version with enhanced features:

  • Scheduled giveaways
  • Advanced entry requirements
  • Custom embed styling
  • Giveaway templates
  • Custom reminders
  • Enhanced analytics
  • Priority support

You can upgrade through our dashboard at any time. Premium is available as a monthly subscription or one-time purchase.

How do I contact support if I need help?

We offer multiple support channels:

  • Join our Discord support server
  • Email us at support@giveawaybot.com
  • Use the /feedback command in your server

Premium users receive priority response times and dedicated support channels.
