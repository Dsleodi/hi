Frequently Asked Questions

Find answers to common questions about Giveaway Pro

How do I install the bot?

Installing Giveaway Pro is simple! Just click the "Invite to Server" button at the top of this page. You'll be redirected to Discord's authorization page where you can select the server you want to add the bot to and configure its permissions.

What permissions does the bot need?

Giveaway Pro requires the following permissions to function properly: View Channels, Send Messages, Embed Links, Add Reactions, and Manage Messages. These permissions are used to create and manage giveaways in your server. The invite link automatically selects the required permissions.

Is there a limit to giveaways?

The free version allows up to 3 concurrent giveaways per server. Premium plans increase this limit to 15 (Pro) or unlimited (Enterprise). There's no limit on the total number of giveaways you can create over time.

How does the bot prevent cheating?

Our advanced anti-cheat system checks for: account age, server join date, participation history, and other behavioral patterns. Premium plans include additional checks like email verification requirements and more sophisticated detection algorithms.

Can I transfer premium between servers?

Premium subscriptions are tied to your Discord account, not individual servers. You can manage which servers have premium features through our dashboard. You can transfer premium benefits between servers at any time.

How do I get support?

You can join our support server through the link in the footer. Free users get community support, while premium users receive priority support with faster response times. For Enterprise customers, we offer dedicated support channels.
